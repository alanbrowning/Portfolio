Vita 3U VPX Software Defined Radio

Professional Project for Collins Aerospace

Details

This project was worked on starting in December of 2021 at Collins Aerospace and is currently ongoing. For this project, I am the lead mechanical design engineer in the development of a 3U VPX software defined radio.

Vita 3U VPX modules are designed in an open system architecture (OSA). OSA essentials boils down to modularity. Each module is considered a line replacable unit (LRU) and will fit into a larger chassis (also defined by VITA). Typically, up to 8 modules can be included in a chassis. Each module will have a particular function. When matched with other modules, complex system functions can be carried out. The main benefit of this type of architecture is high customization for the customer, despite each module being created "standard". Modules can be mixed, matched, and replaced by different vendors to suit the particular needs of the customer.

Collins Aerospace decided to pursue this type of technology by designing a software defined radio which fulfills the VITA 3U VPX requirements. This effort poses many challenges, including thermal, vibrational, size/weight, RF/EMI, and cost constraints. To design this radio, I had to perform structural, thermal, and vibrational analyses via ANSYS and hand calculations. Another key consideration was size. The 3U guidelines are very strict on sizing on components. Detailed tolerance analyses were completed to ensure the fabricated parts were under the maximum size constraints. Additionally, electrical interconnect had to be validated to ensure proper pinout and mating. Although I can't discuss the particular details of this project, my biggest takeaway is that the program demanded constant and clear collaboration between mechanical, electrical, and systems teams. Ignoring the mechanical obstacles, clear communication was critical in designing successful 3U VPX software defined radio.
