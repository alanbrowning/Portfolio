CFD Pipe Flow

Academic Project at the University of Pittsburgh

Details

This project was completed as a part of my applied fluid dynamics course at the University of Pittsburgh. The goal of the project was to teach me how to model fluid flow using Ansys Fluent. Using basic fluid mechanics, you could determine a fully developed velocity profile based on a few properties. This project sought to computationally create this velocity profile as well as determine the entrance lengths based on the Reynold's number. The flow was pressure-driven, and it was assumed to have no thermal effects.

One of the most important factors in a CFD is the mesh. The mesh can entirely change the results of the data, so it is important to create a quality mesh. For standard pipe flow, the mesh does not have to be complicated, but it still needed to have sufficient sizing and wall layers. A picture of the generated mesh is shown below.



The second factor considered involved boundary conditions and material properties. These were given, so no additional work was necessary.

The third factor considered was the minimum acceptable residuals. If too high, the solution could converge early and yield inaccurate results. An appropriate amount was selected at 0.05 [mm/s]. Using these three considerations, the analysis was completed and the results are shown below. The velocity profiles were plotted for a Reynold's number of 50. The entrance lengths were also recorded based on a changing Reynold's number. From the plots, it is clear that the analysis was accurate compared to the hand calculations.
