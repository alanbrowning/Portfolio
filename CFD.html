CFD Pipe Flow

Academic Project at the University of Pittsburgh

Background

This project was completed as a part of my applied fluid dynamics course at the University of Pittsburgh. The goal of this project was to model fluid flow using Ansys Fluent.

Pictures

Details

This project consisted entirely on CFD simulations to accurately represent pipe flow. Using basic fluid mechanics, you could determine the fully developed velocity profile based off of a few properties. This project sought to computationally create this velocity profile as well as determine the entrance lengths based on the Reynold's number. The flow was pressure driven, and it was assumed to have no thermal effects.

One of the most important factors in a CFD is the mesh. The mesh can entirely change the results of the data, so it is important to create a quality mesh. For standard pipe flow, the mesh does not have to be super complicated, but it still needed to have sufficient sizing and wall layers. A picture of the generated mesh is shown below.



The second factor that needed to be considered was the boundary conditions and material properties. These were given, so no additional work was necessary.

The third factor that was considered were the residuals. If too high, it could converge early and yield inaccurate results. An appropriate ammount was selected at 0.05 [mm/s]. Using these three considerations, the analysis was complete and the results are shown below. The velocities profiles were plotted for a Reynold's number of 50. The entrance lengths were also recorded based on a changing Reynold's number. From the plots, it is clear that the analysis was fairly accurate compared to the hand calculations.
