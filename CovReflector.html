Integrated Reflector Selector

Professional Project for Covestro LLC

Details

This project was completed in my second co-op rotation at Covestro LLC. This project revolved around an integrated reflector. This component is a molded heatsink and reflector for an LED. The component removes the necessity for fasteners, and streamlines the production process by molding directly around the LED/reflector. Integrated reflectors were a fairly common product, but the loadcases were often significantly different and required research/testing to find an optimal design. To ease in this process, I completed CFD simulations to help predict how to design these parts based on a desired criteria.

There are many design variables that affect the performance of the reflector. For example, number of fins, fin length, LED power, and material properties all play a huge role. To streamline the process of selecting the optimal characteristics, I completed hundreds of CFD simluations. These CFDs all had slight variations in design variables. For example, I ran some with number of fins varyiing, but all other variables were held constant. Using this process, I calculated a dataset for each variable. The data was fed through a neural network, which helped to determine the optimal combination based on desired criteria (i.e. max junction temp or power dissipation). To further strengthen the validity, I conducted further CFDs to help fill areas that lacking sufficient data. This process was repeated numerous times. In the end, an internal selector tool was created, which allowed the user to set a desired criteria, and the network would recommend the design. This tool was undergoing initial rollout during my final rotation at Covestro.
