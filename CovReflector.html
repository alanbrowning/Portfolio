Integrated Reflector Selector

Professional Project for Covestro LLC

Background

This project was completed in my second co-op rotation at Covestro LLC. Covestro specializes in polymers, and I worked in the business unit tailored specifically to polycarbonates. In this business unit, I worked under the application development team to analyze the viability of polycarbonates within new applications. This project revolved around an integrated reflector. This component is a molded heatsink and reflector for an LED. The design of these components can vary greatly. Typically, substantial research and testing is required to find the optimal design of these components. To ease in this process, I completed CFDs to help predict how to design these parts based on a desired criteria.

Pictures

Unfortunately, this project is on-going, so pictures and results are not available to be shared.


Details

There are many design variables that affect the performance of the reflector. For example, number of fins, fin length, LED power, and material properties all play a huge role. To streamline the process of selecting the optimal characteristics, I completed hundreds of CFDs. These CFDs all had slight variations in design variables. For example, I ran soome with number of fins held constant, but all other variables were independent. Using this process, I calculated lots of data on various combinations. The data was fed through a neural network, which helped to determine the optimal combination based in desired criteria. To further strengthen the validity, I conducted further CFDs to help fill areas that needed more data. This process was repeated numerous times. In the end, a selector tool was created, which allowed the user to set a desired criteria, and the network would recommend the design.
