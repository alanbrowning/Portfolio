Integrated Reflector Selector

Professional Project for Covestro LLC

Details

This project was completed in my second co-op rotation at Covestro LLC. This project revolved around an integrated reflector. This component is a molded heatsink and reflector for an LED. The component removes the necessity for fasteners and streamlines the production process by molding directly around the LED/reflector. Integrated reflectors were a fairly common product, but the load cases were often significantly different and required research/testing to find an optimal design. I completed CFD simulations to help predict the optimal design of these parts based on a desired input.

Many design variables affect the performance of the reflector. Number of fins, fin length, LED power, and material properties all play a significant role. To streamline the process of selecting optimal characteristics, I completed hundreds of CFD simulations. These CFDs all had slight variations in design variables. For example, I ran some with the number of fins varying and all other variables held constant. Using this process, I calculated a dataset for each variable. The data was fed through a neural network, which helped to determine the optimal combination based on desired criteria (i.e. max junction temp or power dissipation). To further strengthen the validity, I conducted additional CFD analyses to help fill areas that lacked sufficient data. This process was repeated numerous times. In the end, an internal selector tool was created, which allowed the user to set a desired input, and the neural network would recommend a design. This tool was undergoing initial rollout during my final rotation at Covestro.
